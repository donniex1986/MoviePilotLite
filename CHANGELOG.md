# Changelog

## 2026-03-31
- 应用设置页新增“更新日志”入口，支持直接查看应用 Changelog。
- Changelog 页面改为 Markdown 渲染，保留标题与列表等格式化展示。
- Changelog 默认仅展开最新版本更新，历史版本改为折叠展开样式以提升可读性。

## 2026-03-30
- 新增 iOS 订阅日历 Widget Extension，支持展示今天及之后的订阅更新。
- 登录态同步到 iOS 共享存储，供 Widget 拉取订阅日历与鉴权图片请求复用。
- 订阅日历 Widget 重构为扁平化卡片样式，按 small、medium、large 尺寸展示不同密度内容。
- small 尺寸调整为“今日订阅概览”，medium 与 large 尺寸优化条目间距、日期信息与图片展示。
- 支持从 iOS 订阅日历 Widget 冷启动进入 App 时直接跳转到应用内订阅日历页面。
- 增加 iOS 专属服务的平台保护，避免非 iOS 平台触发共享会话与 Widget 导航异常。
- 重构媒体整理页布局：引入底部浮动搜索/筛选/排序栏，统一列表筛选入口与交互层级。
- 修复媒体整理页成功/失败筛选逻辑错误，改为基于整理状态进行准确过滤。
- 新增媒体整理结果按时间、标题、大小排序能力，并保留关键词搜索状态。
- 恢复根据标题快速选择整理记录的快捷入口，便于按剧集或条目快速定位。
- 重构站点资源页布局：引入底部浮动搜索/筛选/排序栏，统一资源浏览与筛选入口。
- 站点资源接口返回项映射为统一搜索结果模型，并复用搜索结果卡片与下载弹层交互。
- 新增站点编辑页与编辑控制器，支持加载单站点详情并提交更新站点配置。
- 站点编辑页认证方式改为单个 `Cupertino` 分段切换，在 `Cookie` 与 `ApiKey/Token` 两组字段间互斥切换展示。

## 2026-03-27
- 新增日志页面，并在设置页增加入口，便于查看系统消息与调试信息。
- 调整 API 客户端日志配置，增强问题排查时的可观测性。
- 修复系统环境配置解析异常，提升设置页与系统消息页的数据解码兼容性。
- 修复部分 TOTP 编辑场景下的细节问题。

## 2026-03-26
- 修复动态表单 `InfoCardRow` 重复声明导致的构建错误（`group` 构造改为独立类型）。
- 站点列表页视觉重构：引入底部浮动筛选/搜索/排序栏，优化卡片样式与信息层级。
- 站点列表支持筛选（全部/正常/失败/缓慢）与排序（优先级/上传量/下载量），并增加隐私模式本地持久化。
- 站点项新增连通性状态呼吸灯，状态基于 `/api/v1/site/statistic` 返回值计算。
- 站点列表加载链路增强容错：主列表成功后即展示，后续用户数据与图标请求失败不阻断页面渲染。
- 修复站点缓存主键冲突（`SiteUserDataCache` 去重并使用更新写入）。
- 过滤掉 `id = -1` 的无效站点。
- 修复站点管理页的响应式构建异常，避免统计状态联动时出现 `Obx` 错误。

## 2026-03-22
- 修复 bug
- 下载管理能力升级：支持 qBittorrent（qB）与 Transmission（TR）种子管理。
- 统一种子列表展示与基础操作（暂停、恢复、删除、重命名、分类/标签、限速）。

## 2026-03-23

- 新增下载任务页面 `torrent_download_screen`，补全 qBittorrent/Transmission 提交逻辑与错误提示。
- 下载管理页支持筛选与排序条件本地持久化恢复，减少页面状态回退。
- 优化种子列表项视觉层级与间距，降低信息拥挤感。
- 调整下载管理入口路由逻辑，支持按设置切换到下载器配置页。
- 修复媒体详情页骨架加载与 `soft_edge_blur` 组合导致的断言问题。
- 移除种子管理本地 Realm 缓存逻辑，回归在线数据流。
- fix: douban media detail subscribe state error
- 新增独立 `altman_totp` package，支持 TOTP 新增、编辑、删除、选择回填登录 OTP。
- TOTP 本地存储改为 `sqflite`，支持多条记录与 UUID 主键。
- 新增 TOTP 扫码录入与手动密钥录入，支持解析 `otpauth://` 并自动回填用户名。
- 登录页 OTP 选择改为 modal 方式拉起 TOTP 选择页。
- TOTP 管理页与编辑页增加验证码倒计时与逆向进度显示。
- TOTP 管理页支持单项导出，提供二维码与字符串（含 `name`、`key`）复制。
- 增加 iOS/Android 相机权限配置以支持二维码扫描。
- tv 类型订阅完成后支持二次编辑
